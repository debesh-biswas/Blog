# Debesh — Personal Blog

I'm Debesh. This repository contains my personal blog and documentation site where I publish blogs, project notes, and technical write-ups. The site is content-first: I use it to capture ideas, experiments, and practical knowledge about machine learning, developer tooling, and documentation practices.

What I focus on
- Personal essays and long-form writing about tooling and ML
- Project notes, design decisions, and implementation rationale
- Practical how-tos and reference material I want to revisit later

Project goals
- Keep content easy to author and maintain
- Prioritize performance, accessibility, and readable documentation
- Iterate on design and content as I learn and experiment

Tech stack
- Astro: content-focused static site framework
- TypeScript: type-safe scripting for site components and tooling
- Node / npm: build tooling and package management
- Netlify: deployment and hosting (configuration in netlify.toml)

Repository layout
- `src/` — site source: pages, components, styles, and content
- `public/` — static assets and theme files
- `scripts/` — maintenance and build scripts
- `package.json`, `astro.config.mjs`, `tsconfig.json` — project configuration
